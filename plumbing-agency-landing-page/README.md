# Plumbing Agency Landing Page

## Project Overview
This project is a landing page for a plumbing agency named QuiCano. It aims to provide potential clients with information about the agency's services, history, and contact details. The website is designed to be user-friendly and visually appealing, encouraging visitors to reach out for plumbing services.

## Project Structure
```
plumbing-agency-landing-page
├── src
│   ├── index.html        # Main landing page
│   ├── servicos.html     # Services offered by the agency
│   ├── sobre.html        # Information about the agency
│   ├── contato.html      # Contact form and details
│   └── styles.css        # CSS styles for the project
├── package.json          # npm configuration file
└── README.md             # Project documentation
```

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd plumbing-agency-landing-page
   ```
3. Install dependencies (if any):
   ```
   npm install
   ```

## Usage
- Open `src/index.html` in a web browser to view the landing page.
- Navigate through the links to explore services, learn about the agency, and access the contact form.
